About Kubetoy 0.3.0

This application is a simple Node.js application that is expected to run in a container deployed by kubernetes. It was created to help explore IBM Cloud Private by allowing the user to write messages to the log, crash the application, and even create and view files in a shared file system.

This app uses Bootstrap 4, and it is expected that the browser will have access to the content delivery network versions of the CSS and JavaScript files.

The app was written by jim conallen (jconallen@us.ibm.com) and is made available under the Apache 2.0 license.
